About TrackCat

TrackCat was first an ice-breaker to coding for a group of girls in 2014. It is a project created through CodeCatz - a coding study group where women are always in majority.

Since the best way to learn is by doing, we set up this page and along the way learned in practice how HTML, CSS and Django work. Now its time for other projects. What we will create next will be seen in our Projects page.
